setLaBrDist [N] [distance]

Set source/origen to detector housing face
[usage] /OCL/det/oscar/setLaBrDist N distance
N:(int) Internal detector (face) number
distance:(double) Distance (in cm)
Note:
- N=0 and N=32 are the beamline facets
- No geometry check is performed, so you're responsible to check
that there will not be any geometry overlap

Available Geant4 state(s) : PreInit

Parameters
N type i Parameter range : N>=1&&N<=30
distance type d Parameter range : distance>=12
